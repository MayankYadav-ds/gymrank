# Exercises Module

Responsibilities:

- Exercise library
- Exercise details
- Approved ranking lift identification
- Exercise variants
- Primary and secondary muscle targeting
- Exercise-centric anatomy data

## Milestone 3 Endpoints

- `GET /v1/exercises`
- `GET /v1/exercises/:id`

Optional query parameters:

- `search`
- `trackedOnly=true`

## Scope Guard

This module does not implement workout logging, PRs, overload recommendations, rankings, analytics, achievements, community, or a full interactive anatomy explorer.
