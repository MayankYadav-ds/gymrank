# Overload Module

Responsibilities:

- Conservative next weight or rep targets
- Insufficient-history states
- Avoid aggressive max-lift encouragement

## Milestone 6 Scope

- Deterministic recommendations from completed workout history
- No AI
- No randomness
- No rankings, analytics, achievements, or PR mutation

## Endpoints

- `GET /v1/overload/recommendations`
- `GET /v1/overload/:exerciseId`
