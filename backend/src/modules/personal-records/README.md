# Personal Records Module

Responsibilities:

- Best set tracking
- PR detection after workout completion
- PR history

## Milestone 5 Scope

- Detect PRs when a workout is completed
- Use completed sets only
- Compare best set by highest weight, with reps as the tie-breaker
- Store PR history
- Return current PRs
- Return per-exercise PR history

## Not Included

- Estimated 1RM
- Rankings
- Progressive overload
- Analytics
- Achievements
