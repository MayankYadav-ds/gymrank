# Workouts Module

Responsibilities:

- Workout sessions
- Exercises within sessions
- Sets, notes, history, edits, and deletes

## Milestone 4 Scope

- Create workout
- Start workout through creation
- Finish workout
- Cancel workout
- Workout history
- Workout detail
- Delete workout
- Add/remove/reorder workout exercises
- Exercise notes
- Add/update/delete unlimited sets

## Not Included

- Workout templates
- Programs
- Personal records
- Progressive overload
- Rankings
- Analytics
- Achievements
