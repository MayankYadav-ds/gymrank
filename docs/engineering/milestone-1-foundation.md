# Implementation Milestone 1: Foundation Setup

## Goal

Create the initial GymRank V1 monorepo foundation for the approved stack:

- Mobile: Flutter
- Backend: Node.js + TypeScript + Express
- Database: PostgreSQL
- ORM: Prisma
- Local DB: Dockerized PostgreSQL
- Storage: S3-compatible object storage later

## Deliverables

- Monorepo folder structure
- Flutter mobile skeleton
- Express + TypeScript backend skeleton
- Dockerized PostgreSQL local setup
- Prisma placeholder schema
- Environment templates
- Linting, formatting, and testing setup
- Approved tracked lifts constants
- Module placeholders for all approved V1 modules

## Non-Goals

- No full auth implementation
- No workout implementation
- No PR engine
- No ranking engine
- No analytics logic
- No community or social features
- No ranking outside approved tracked lifts
