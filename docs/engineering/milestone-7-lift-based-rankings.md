# Implementation Milestone 7: Lift-Based Rankings

## Scope

Milestone 7 implements GymRank V1 lift-based rankings only.

## Included

- Ranking snapshots sourced from Personal Records
- Per-lift leaderboards
- Current user's ranking
- Nearby rankings
- Pagination
- Country, sex/category, and bodyweight class filters
- Configurable bodyweight classes
- Mobile leaderboard, my rank, ranking card, filter sheet, and nearby list skeletons

## Not Included

- Analytics
- Achievements
- Community
- Social feed
- Messaging
- AI
- Nutrition
- Programs
- Templates
- Estimated 1RM
- Wearables
- Notifications

## Ranking Rule

Rank priority:

1. Highest weight
2. Highest reps
3. Earlier achievement date

Only approved V1 tracked lifts can appear in rankings.
