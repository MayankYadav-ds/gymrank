# GymRank V1 Scope

GymRank V1 is a competitive strength progress platform.

## Core Loop

```text
log workout -> detect PRs -> recommend overload -> update rankings -> show progress analytics
```

## Included

- Authentication and profile
- Workout tracking
- Exercise library
- Per-exercise muscle targeting display
- Personal records tracking
- Progressive overload recommendations
- Lift-based rankings for approved tracked lifts only
- Basic physique tracking
- Basic analytics dashboard
- Basic achievements

## Ranking Lifts

V1 rankings are restricted to:

- Bench Press
- Squat
- Deadlift
- Overhead Press
- Pull-Up / Weighted Pull-Up

## Excluded From V1

- Community feed
- Comments, likes, follows
- Public PR sharing
- Full social posting system
- Full interactive tappable anatomy explorer
- Advanced physique analytics
- AI coaching
- Direct messages
- Video content
- Advanced challenges
- Creator or coach tools
