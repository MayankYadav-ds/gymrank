# Approved V1 Tracked Lifts

Only these lifts are eligible for GymRank V1 rankings:

- `bench_press` - Bench Press
- `squat` - Squat
- `deadlift` - Deadlift
- `overhead_press` - Overhead Press
- `pull_up_weighted_pull_up` - Pull-Up / Weighted Pull-Up

The full exercise library must not be globally ranked in V1.
